# Desafio Disce Youbiz

Implemente um CRUD simples usando a API do JSONPlaceholder.

## API
Use a API disponível em: https://jsonplaceholder.typicode.com/guide/

Nota: Esta é uma API falsa (JSONPlaceholder) e as alterações não são persistidas no servidor.

## Requisitos
- Implementar página para ler post selecionado ( /posts/:id )
- Criar operações CRUD (Create, Read, Update, Delete) para posts.
- Implemente uma interface simples para interagir com a API.
- Utilize Tailwind CSS para estilização. A interface será avaliada.

## Funcionalidades Opcionais
- Filtragem de dados (por exemplo, filtrar posts por usuário ou título).
- Mensagens toast para feedback das ações

## Como começar

Clone o repositório:

```bash
git clone <url-do-repo>
cd desafio_disce_youbiz
```

## Como executar

1. Instale as dependências: `yarn install`
2. Execute o servidor de desenvolvimento: `yarn dev`
3. Abra [http://localhost:3000](http://localhost:3000) no navegador.

## Avaliação

Após implementar, faça o repositório público no GitHub para avaliação.
